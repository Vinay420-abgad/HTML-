Information about c++

C++ is a high-level, general-purpose programming language developed by Bjarne Stroustrup in 1979. It supports procedural, object-oriented, and generic programming, making it fast, powerful, and portable. Commonly used in system software, game development, and applications.


★ Key Features of c++ :-

1. Simple & Efficient – Easy to learn (if you know C) and runs fast.

2. Object-Oriented – Supports classes, objects, inheritance, polymorphism, encapsulation.

3. Portable – Can run the same code on different platforms.

4. Rich Library (STL) – Provides built-in data structures and algorithms.

5. Memory Management – Allows low-level memory access with pointers.

6. Multi-paradigm – Supports procedural, object-oriented, and generic programming.

7. High Performance – Used for speed-critical applications (games, OS, compilers).
