Information About Cascading Style Sheets (CSS)

CSS is a style sheet language used for describing the presentation of a document written in a markup language like HTML or XML. It is one of the three core technologies of the World Wide Web, alongside HTML for structure and JavaScript for interactivity.

Key aspects of CSS include :-

Seperations of Concerns :
CSS allows for the separation of content (HTML) from its presentation (CSS). This makes web development more organized, efficient, and easier to maintain.

Styling Web Pages :
CSS is used to control the visual appearance of web pages, including elements such a colors, fonts, spacing, layout, and even animations.

Cascading Nature :
The Cascading in CSS refers to the set of rules that determine which styles are applied when multiple styles conflict. This system prioritizes styles based on factors like specificity, importance, and source order.

Syntax:
CSS consists of rules that are applied to elements using selectors. A rule typically includes a selector (e.g., p for paragraphs, .my-class for elements with a specific class) and a declaration block containing one or more declarations. Each declaration consists of a property (e.g., color, font-size) and its corresponding value (e.g., blue, 16px).

Advantages:
CSS offers numerous benefits, including faster website loading times due to smaller file sizes, improved consistency across multiple pages, easier maintenance and updates, and enhanced flexibility in design.

Usage:
CSS can be applied to html document in several ways :
  • External Style Sheets : Linking a .css file to the html document.
  • Internal Style Sheets : Embedded CSS within Style tags in html document's Head.
  • Inline Styles : Applying styles directly to individual HTML elements using the style attribute.
